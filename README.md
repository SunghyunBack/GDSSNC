# 🏢 GDSSNC (2023.11 ~ 2025.12)

**포지션:** C# / .NET 응용 프로그램 개발자 (BIM 설계 자동화 파트)
**핵심 기술:** `C#`, `.NET`, `Revit API`, `AutoCAD API`, `DevExpress`, `TortoiseSVN`

## 💡 주요 업무 개요
건축/건설 도메인의 설계 생산성을 극대화하는 **BIM 3D 설계 자동화 솔루션**을 전담하여 개발했습니다.
단순 반복적인 수작업을 시스템화하는 것을 넘어, **최단 경로 탐색 로직 적용, 유체 역학 수식의 전산화, 트랜잭션 최적화** 등 소프트웨어 공학적 접근을 통해 설계 파이프라인을 혁신하고 실무자의 작업 시간을 획기적으로 단축하는 데 기여했습니다.

## 📈 핵심 역량 및 엔지니어링 철학

- **시스템 성능 및 메모리 최적화 (Performance Optimization)**
  - 무거운 3D 도면(BIM) 환경에서 Transaction 호출을 최소화하고, 파편화된 변수를 캡슐화(리팩토링)하여 렌더링 딜레이를 해소하는 등 시스템의 안정성을 설계 단계부터 고려합니다.
- **복잡한 비즈니스 룰의 시스템화 (Domain Integration)**
  - 단순한 코딩을 넘어 건축 법규, 유체 역학 공식 등 복잡한 도메인 지식을 이해하고, 이를 조건화하여 100% 자동화된 소프트웨어 파이프라인으로 풀어내는 역량이 있습니다.
- **실무자 중심의 유연한 구조 설계 (User-Centric UX & Maintainability)**
  - 예외 상황(Edge Case)에 대비한 동적 노드 편집 UI 구현, 타인의 레거시 코드 리팩토링 등 '실제 현장에서 쓰기 편하고 유지보수가 용이한 제품'을 만드는 데 집중합니다.
