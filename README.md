# 🏢 GDSSNC (2023.11 ~ 2025.12)

**포지션:** C#/.NET 응용 프로그램 개발자
**기술 스택:** `C#`, `.NET`, `Revit API`, `DevExpress`, `AutoCAD`

## 💡 주요 업무 개요
건축 및 건설 도메인의 설계 효율성을 극대화하기 위한 **BIM 설계 자동화 프로그램**을 개발하고 있습니다. 
Revit 환경에서 반복적으로 발생하는 설계(설비, 전기, 건축 등) 작업을 자동화하고, 알고리즘을 적용해 최적의 설계 경로를 도출하는 비즈니스 로직을 구현하여 실무자의 작업 시간을 단축하는 데 기여하고 있습니다.

## 📈 배운 점 및 핵심 역량
- **성능 최적화 경험:** 무거운 3D 도면(BIM) 환경에서 API 호출을 최소화하고 트랜잭션을 효율적으로 관리하는 방법을 깊이 고민하고 적용했습니다.
- **도메인 지식의 융합:** 단순한 코딩을 넘어 건축/설비의 비즈니스 로직을 이해하고, 이를 소프트웨어 알고리즘으로 풀어내는 엔지니어링 역량을 길렀습니다.
- **소프트웨어 완성도 추구:** 복잡한 조건 속에서도 예외 상황(Edge Case)을 방지하기 위해 꼼꼼하게 점검하고 트러블슈팅하는 대응력을 갖추었습니다.
