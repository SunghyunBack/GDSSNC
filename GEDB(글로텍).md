# 📊 GEDB: Wijmo Grid 기반 고성능 데이터베이스 관리 및 조회 시스템

## 1. 프로젝트 개요 및 비즈니스 임팩트
- **기간:** 2025.05.19 ~ 2025.06.04
- **인원:** 1명 (단독 프로젝트)
- **목적 및 성과:** - 대용량 데이터를 효율적으로 관리하고 시각화하기 위한 전사적 데이터베이스 관리 시스템(GEDB)을 구축함.
  - **Wijmo Grid**를 활용하여 수만 건 이상의 데이터를 지연 없이 렌더링하고, 엑셀과의 상호운용성을 확보하여 현업의 데이터 가공 업무 시간을 획기적으로 단축함.

---

## 2. 기술 스택
- **Frontend:** `React`, `TypeScript`, `Material-UI`, `Wijmo Grid`
- **Backend:** `Node.js`, `Express.js`, `Passport.js` (Auth)
- **Database:** `PostgreSQL`
- **Tool:** `Cursor` (AI-powered IDE)

---

## 3. 핵심 기능 및 아키텍처

### 🔹 고성능 데이터 그리드 시스템 (Wijmo Grid)
- **대용량 처리:** Wijmo Grid의 가상화 기술을 적용하여 수천 개 이상의 로우(Row)를 웹 환경에서 끊김 없이 표시 및 정렬합니다.
- **Excel Interoperability:** 데이터 그리드의 내용을 Excel로 내보내거나(Export), 외부 Excel 파일을 가져와(Import) DB에 즉각 반영하는 데이터 동기화 기능을 구현했습니다.
- **사용자 맞춤형 필터링:** Material-UI와 연동하여 다중 조건 필터링 및 복합 정렬 기능을 제공합니다.

### 🔹 안정적인 세션 기반 보안 인증 (Auth)
- **Passport.js:** 세션 방식의 인증 시스템을 구축하여 사용자별 접근 권한을 관리하고 보안성을 강화했습니다.
- **State Management:** React 환경에서 인증 상태를 효율적으로 관리하여 페이지 접근 제어 및 사용자 경험을 개선했습니다.

---


## 4. 향후 확장 계획
* **QA 자동화:** Jest를 활용한 단위 테스트 및 통합 테스트 코드를 강화하여 시스템 안정도 제고.
* **성능 모니터링:** Prometheus 및 Grafana 연동을 통한 실시간 서버 부하 모니터링 시스템 도입 예정.
* **UX 개선:** 사용자 접근성을 고려한 UI 컴포넌트 고도화 및 반응형 웹 최적화.
